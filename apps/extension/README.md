# LoopGuard — VS Code Extension

**Stop wasting hours on AI-assisted loops. LoopGuard detects when you're stuck and breaks the cycle.**

## What it does

- **Loop Detection** — Identifies when you're repeating the same AI-assisted fix with no progress. Shows you how much time you've wasted.
- **Context Engine** — Filters and selects only the relevant code context before AI calls. Reduces token waste by 80–98%.

## Features

- Real-time loop detection via VS Code diagnostics
- Session time tracking — see exactly how long you've been stuck
- Status bar alerts with time-wasted counter
- Fast Rust-powered context engine (89–99% token reduction)
- TypeScript fallback engine (~80% reduction) — works on all platforms

## Requirements

- VS Code 1.85.0 or later
- A free LoopGuard account (sign up at [loopguard.vercel.app](https://loopguard.vercel.app))

## Getting Started

1. Install the extension
2. Open any project in VS Code
3. Click the LoopGuard icon in the status bar to sign in
4. Start coding — LoopGuard runs silently in the background

## Privacy

**Your code never leaves your machine.** LoopGuard only sends anonymized metrics (loop counts, session durations) to the backend. All analysis runs locally.
